## 🚀 Descripción

IA Pro 2026 es una plataforma educativa interactiva construida como entorno de estudio autodidacta profesional para aprender:

* IA Generativa
* Prompt Engineering
* APIs de IA
* Automatización No-Code
* Make.com
* n8n
* Agentes IA
* Workflows autónomos
* RAG y memoria
* Integraciones modernas
* Sistemas IA productivos reales

El objetivo es ofrecer una experiencia equivalente o superior a formaciones premium, utilizando recursos gratuitos y prácticas aplicadas.

## ✨ Funcionalidades

### 📚 Plataforma educativa completa

* Unidades progresivas

### 🧠 Aprendizaje práctico

* Ejercicios hands-on
* Casos reales
* Automatizaciones aplicadas
* Integraciones con APIs
* Agentes IA funcionales

### 💾 Persistencia local

La plataforma guarda automáticamente:

* progreso,
* notas,
* favoritos,
* estado de módulos,

utilizando `localStorage`.
