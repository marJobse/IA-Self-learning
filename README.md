## 🚀 Descripción

IA Pro 2026 es una plataforma educativa interactiva construida como entorno de estudio autodidacta profesional para aprender IA generativa, prompting, APIs, automatización no-code, Make, n8n, agentes IA, RAG y workflows aplicados.

## ✨ Funcionalidades

- Roadmap estructurado por módulos.
- Dashboard de progreso.
- Unidades progresivas.
- Recursos externos curados.
- Ejercicios prácticos.
- Proyectos integradores.
- Favoritos.
- Notas personales.
- Persistencia local con `localStorage`.

## 🛠️ Stack tecnológico

- React
- Vite
- JavaScript / JSX
- CSS3
- LocalStorage API
- Tabler Icons
- Vercel

## ⚙️ Uso local

```bash
npm install
npm run dev
```

Para generar versión de producción:

```bash
npm run build
```

## 🚀 Deploy en Vercel

Vercel detecta automáticamente Vite. Configuración esperada:

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
