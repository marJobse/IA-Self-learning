# 🚀 Descripción

IA Pro 2026 es una plataforma educativa interactiva desarrollada como entorno de aprendizaje autodidacta profesional para estudiar:

* IA Generativa
* Prompt Engineering
* APIs de IA
* Automatización No-Code
* Make.com
* n8n
* Agentes IA
* Workflows autónomos
* RAG y memoria
* Integraciones modernas
* Sistemas IA productivos reales

El objetivo del proyecto es construir una experiencia educativa moderna, práctica y escalable, equivalente o superior a muchas formaciones premium.

# ⚡ Instalación local

## Clonar repositorio

```bash
git clone https://github.com/marJobse/IA-Self-learning.git
```

---

## Instalar dependencias

```bash
npm install
```

---

## Ejecutar entorno de desarrollo

```bash
npm run dev
```

---

## Build de producción

```bash
npm run build
```
